# Dataset

Run `python data/cached_fineweb10B.py 40` to download the FineWeb dataset.

# Scripts

We provide two example scripts:
- `compare_base.sh` to compare the performance of EMA-Nesterov on different base optimizers (Muon, NorMuon, SOAP, Adam).
- `compare_lookahead_muon.sh` to compare the performance of different lookahead algorithms on Muon.
